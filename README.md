# Probabilistic Modeling of Latent Swing Quality and Contact Efficiency Using Public MLB Statcast Bat-Tracking Data

Author: Jiho Choi
Program: Master of Science in Data Science and Analytics
Supervisor: Dr. Pralat
Institution: Toronto Metropolitan University
Season: 2025 MLB Regular Season

This repository contains the code, data, and outputs for a Master's Major Research Project (MRP) investigating whether publicly available MLB bat-tracking metrics can be used to construct a reliable, probabilistic estimate of latent swing quality at the player level.

The project addresses three research questions:
RQ1: To what extent do bat-tracking variables (avg_bat_speed, ideal_attack_angle_rate) explain contact efficiency and expected offensive value (xwOBA, barrel rate, hard hit rate)?
RQ2: Can latent swing quality be estimated probabilistically after accounting for sample-size uncertainty and data reliability?
RQ3: How can player-level swing-quality profiles be monitored and compared across player groups using an interactive visualization interface?

Important note: All analyses use publicly available, summary-level season-aggregate data from Baseball Savant. This is not raw pitch-by-pitch swing trajectory data.
